**TODO-APP-MWINZA**  
*HOW TO RUN*
1. Install nodejs by visiting [nodejs website](https://nodejs.org/en/download) and following download and installation instructions for your operating system.  
2. Open your terminal and run `npm install -g ts-node typescript` to install type script.
3. In your terminal, run `ts-node <path to project directory>/src/todo-app-mwinza.ts`
